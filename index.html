포커스

레이어가 나타난 상태에서의 포커스 순서가 논리적이지 않은 경우

오류 예시
전체 카테고리메뉴 레이어가 나타난 상태에서의 포커스 순서가 '전체카테고리 열기 아이콘 → 뮤지컬 → 콘서트 → 연극 → ...' 순으로 구성되어 논리적이지 않음
참고 이미지
해결 방법

'전체카테고리 열기 아이콘 → 오리지널/내한공연 → 라이선스 → 창작뮤지컬 → ...'순으로 구성되도록 수정되어야 함

자동으로 갱신되는 콘텐츠의 제어

키보드 포커스 시 정지되지 않는 경우

오류 예시
키보드 포커스시 롤링되는 콘텐츠 내용이 지속적으로 정지 해있지 않음.
참고 이미지
해결 방법

키보드 포커스 시 정지 또는 정지 버튼 제공

적절한 링크 텍스트

목적이나 용도를 알기 어려운 링크 텍스트를 제공한 경우

오류 예시
쎈DEAL 더보기를 그냥 '더보기'로만 제공하여 링크 텍스트가 단독으로 사용될 때 정확한 의미 파악이 불가능하다.
참고 이미지
해결 방법

'쎈DEAL 더보기'란 텍스트를 제공해야함. '쎈DEAL'을 숨김텍스트로 제공해주거나 title로 전체 링크 텍스트를 제공해야한다. 단, title 속성은 보조 도구의 설정에 따라 사용자에게 전달되지 못하는 경우도 발생할 수 있으므로 숨김 텍스트로 제공하는 것을 권장함.

이미지 대체텍스트

하나의 링크내에 설명문구와 이미지의 대체 텍스트를 중복해서 제공한 경우

오류 예시
"★무료배송★ 로지텍코리아 K480 멀티 블루투스 키보드 (블랙/화이트)"라는 항목 이미지의 대체 텍스트가 함께 제공된 텍스트와 중복됨.
참고 이미지
해결 방법

하나의 링크내에 설명문구와 이미지의 대체 텍스트를 중복 제공한 경우 이미지의 alt값을 빈값(alt="")으로 제공해야 함.
하단에 제공된 사례와 같이 이미지와 텍스트에 각각의 링크가 걸려있는 경우, 링크가 이미지와 텍스트를 한꺼번에 감싸도록 수정하는 것이 바람직함.

이미지의 내용중 일부만 대체 텍스트로 제공한 경우

오류 예시
이미지의 내용중 일부만 대체 텍스트로 제공한 경우
참고 이미지
해결 방법
